🎨 PortableUi Layout System Demo

1. Basic Container Layout (Horizontal)

A basic horizontal container with gap and padding. Child components are arranged side-by-side.

2. Vertical Container Layout

A vertical container stacking components vertically with customizable gap and alignment.

3. Flex Layout (Flexible Distribution)

Demonstrates flex containers with different grow/shrink ratios. The middle box has 2x the space of others.

4. Grid Layout (3-Column Grid)

A 3-column grid layout. First item spans 2 columns, demonstrating grid item spanning.

5. Group Layout (With Title and Border)

Grouped form with sections. Shows how to organize related elements with titles and borders.

6. Complex Nested Layout

A complete dashboard-like layout combining Container, Flex, Grid, and Group components.

7. Responsive Flex Wrap

Cards layout with flex-wrap enabled. Components wrap to next line when space is constrained.
